User Guide

Add Diary Entry

In the home page tap on the location pin icon to get your current location. When prompted enter a Title (Required) and a Description (Optional) and save the location in Geo Diary.

View Diary Entries

From the side menu tap on the ‘My Locations’ menu item to view all Geo Diary entries. Tap on one entry and move to the details page to see full details including the Title, Description (if any), Time stamp and the Location is pined to a Google Map where you have the full control to view it.

Search Diary Entries

In ‘My Location’ use the search box on the top of the page to search Geo Diary entries by Title to quickly find your diary entries.
